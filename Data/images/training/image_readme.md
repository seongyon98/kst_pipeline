# AI Hub 수학 과목 자동 풀이 데이터

## Question BBox 좌표 해석
- **[x, y, w, h]**
  - `x`: 크롭할 영역의 왼쪽 상단 모서리의 X 좌표 (가로 위치).
  - `y`: 크롭할 영역의 왼쪽 상단 모서리의 Y 좌표 (세로 위치).
  - `w`: 크롭할 영역의 너비(Width).
  - `h`: 크롭할 영역의 높이(Height).
- **line**: 텍스트 한 줄 영역 ("한 개 이상").
- **paragraph**: line들의 집합 영역.
- **image**: 이미지 영역 (figure_text: image 영역에 대한 캡션).
